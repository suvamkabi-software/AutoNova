# 🚗 AutoNova – Modern Car Showcase Website

AutoNova is a modern, fully responsive car showcase website built using pure HTML and CSS. It provides a premium user experience with interactive UI elements, smooth navigation, and a sleek automotive design.

---

## 🌟 Features

- 🔥 Fully responsive (Mobile + Desktop)
- 🎨 Modern UI/UX with dark theme
- 🚗 Car models categorized (Sports, SUVs, Electric)
- 🎯 Interactive tabs and dropdown menus
- ⚙️ Car configurator (color preview using CSS)
- 📊 Performance ratings using meter bars
- 📩 Contact form with modal popup
- 📌 Smooth scrolling navigation
- ❌ No JavaScript (Pure HTML + CSS)

---
## 📚 Purpose of This Project

This project was created as a **practice project** to improve my frontend development skills using pure HTML and CSS.

The goal was to:
- Learn responsive design
- Practice modern UI/UX layouts
- Build interactive components without JavaScript
- Understand CSS Flexbox, Grid, and animations

---

## 🎯 Learning Outcome

Through this project, I gained hands-on experience in:

- Structuring large HTML projects
- Writing clean and maintainable CSS
- Creating interactive UI using only HTML & CSS
- Designing real-world website layouts
